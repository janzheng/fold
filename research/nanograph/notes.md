# Research Notes: nanograph--nanograph

## Source

- **Repo:** [nanograph/nanograph](https://github.com/nanograph/nanograph)

## What it is

On-device typed property graph database built on Rust/Lance/Arrow. Schema-as-code (.pg files), full search stack (BM25/semantic/hybrid), built-in CDC, agent-native with no server required.

## What to study

- Schema-as-code pattern (.pg files)
- Agent-native database design (no server)
- Hybrid search stack (BM25 + semantic + hybrid)
- Rust/Lance/Arrow architecture for local-first data
